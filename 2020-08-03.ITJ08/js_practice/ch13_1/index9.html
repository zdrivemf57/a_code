Javaでの記述

public class Member() {
  // フィールド
  public String name;
  public int age;

  // コンストラクタ
  public Member(name, age) {
    this.name = name;
    this.age = age;
  }

  // メソッド
  public void showInfo() {
    System.out.println(name +  "[" + age + "]");
  }
}

-----
クラスの利用
Member m = new Member("山田", 25);
m.showInfo(); // 山田[25] と表示
