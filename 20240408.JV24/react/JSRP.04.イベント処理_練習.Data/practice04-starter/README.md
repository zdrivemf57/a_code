# 練習問題テンプレート

- 練習問題のテンプレートとなるフォルダです。
- `create-react-app`の後、不要なファイルを削除したものになります。

## 作業開始手順

1. このテンプレートフォルダを VS Code で開く
1. `Ctrl + @`でターミナルを開き、`npm install`
1. `npm start`でサーバーを起動する
